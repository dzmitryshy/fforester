== Description ==
Photo Video Store theme is based on Free Corlate theme (https://github.com/jtherczeg/corlate-theme). It uses Bootstrap 3 css framework.
You can sell photos online with Photo Video Store Wordpress plugin, create Stock Photography blog, run Shopping cart for Selling Prints.

== Installation ==

1. Download the theme at https://www.cmsaccount.com/templates/
2. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
3. Select a theme's zip archive and click "Install".
4. Click on the 'Activate' button to use your new theme right away.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.

== Blog ==

The theme includes a blog page. To activate it you should create a new page in Admin panel -> Pages -> Add new and select 'Blog' page template in the right menu.

== Menu ==

To change a menu you should use Appearance > Menus.

Also you can show a default PVS menu (file menu.php) by removing comment from the line:
get_template_part('menu');
in header.php file.

and add a comment for the line:
//wp_nav_menu($args);
